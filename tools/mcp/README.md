# MCP Tools

MCP (Model Context Protocol) server tool definitions for use with Claude Code and compatible agents.

## Structure

Each tool is a folder:
```
tools/mcp/<tool-name>/
├── server.py       # MCP server implementation
├── README.md       # Tool description and usage
└── config.json     # Tool configuration
```
